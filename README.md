#ArrowCounter

## Description

Simple mod for Hytale game. shows a little cutyy ui for knowing how many arrows you still have. omg it is so close to zero time to craft some back !!!

## Downloading

Download the arrowcounter_x.x.x.jar file either from modtale, curseforge, or direclty from github releases : https://github.com/louisaiva/arrow-counter/releases

## Installation

1. Move the downloaded jar file into your Hytale mods folder, usually located at:
`C:\Users\<YourUsername>\AppData\Roaming\Hytale\UserData\Mods`

2. Then launch Hytale

3. Right click on your world and ensure that "ArrowCounter" mod is enabled

4. All done ! Omg using the bow is so much better now !!!
